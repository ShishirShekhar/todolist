# todolist

## 📃 Intro

🎯 Goal: This website is a to do list app.

🎖 Feature: 
- You can add task you want to do
- Delete them after they are done
- You can create different pages for different types of tasks


## ‼ Dependencies
    "body-parser": "^1.19.1",
    "dotenv": "^14.3.2",
    "ejs": "^3.1.6",
    "express": "^4.17.2",
    "lodash": "^4.17.21",
    "mongoose": "^6.1.8"


## 👩‍💻 Install Dependencies
> npm i


## 🏃‍♂️ Run app
> node app.js
